## 🌟 Overview

**JasVolunteer.kz** is a strategic solution designed to digitize and unify the volunteer movement in Kazakhstan. It transforms fragmented charitable actions into a transparent, data-driven system where every contribution is tracked, verified, and rewarded.

### Key Solutions:

  * **For Organizations (NGOs):** A centralized dashboard to publish social projects, verify legitimacy via BIN (Business Identification Number), manage volunteer pipelines, and access impact analytics.
  * **For Volunteers:** A "Digital Volunteer Passport" featuring participation history, verified hours, a ranking system, and access to a curated list of opportunities in Almaty and across the country.

## 🛠 Tech Stack

### **Frontend**

  * **Framework:** Angular 17+ (TypeScript)
  * **Styling:** Tailwind CSS / SCSS
  * **State Management:** RxJS

### **Backend**

  * **Framework:** Django 5.0 & Django REST Framework (DRF)
  * **Authentication:** JWT (JSON Web Tokens)
  * **Database:** PostgreSQL
